Lier un fichier JavaScript et un fichier CSS /1

  • Liez le fichier style.css qui se trouve dans le dossier styles à cette page.
  • Créez un fichier main.js dans le dossier scripts et liez le à cette page HTML.
  • Faire apparaître dans la console le message "Fichier main.js en place."
  • Remplacer ce texte par : "Texte remplacé via main.js"

Si vous n'y arrivez pas, demandez-moi pour pouvoir avancer sur la suite des exercices.

Manipuler une variable simple /2

  • Créez une variable "compteur" initialisée à 0 dans votre main.js
  • Affichez dans la div ci-dessous un élément qui affiche la valeur de ce compteur
  • Créez un bouton dans la div ci-dessous qui permet de faire augmenter ce compteur de 1 à chaque clique
  • Faire en sorte que le compteur retourne à zéro tous les 10 coups
    Faire en sorte qu'après le 10, on retourne à 1...

Animation au clic /3

  • Depuis le fichier style.css et le fichier main.js, faites en sorte qu'au clic sur le bouton suivant, le fond de cette section passe du blanc au gris foncé et que le texte passe du noir au blanc.
  • Remplacer en même temps le texte du bouton "mode jour" (quand on est en mode nuit) et "mode nuit" (quand on est en mode jour)
  • A chaque clic on doit pouvoir passer d'un mode à l'autre
  • Le changement doit se faire avec une transition.

Utiliser un tableau simple /3

  • Lier le fichier exercice_tableau.js du dossier scripts à cette page
  • En utilisant le "fruits_disponibles" fournit dans ce fichier, Afficher dans l'ul ci-dessous la liste des fruits disponibles.
  • Depuis ce même fichier, faire une fonction pour récupérer la valeur saisie dans l'input "fruit_desire" au clic sur le bouton "valid_choice" et vérifier si le fruit est présent parmi les fruits disponibles.
    • Afficher "disponible" dans message_exercice_fruit si présent.
    • Afficher "non disponible" si non présent.

Reproduire la structure ci-dessous /3

  • Lier le fichier exercice_actus.css du dossier styles à cette page.
  • Mettre le CSS nécessaire à la reproduction de l'image à l'intérieur de ce fichier.
    Mettez les typos et les couleurs que vous voulez. Les images sont dans le dossier images.

Formulaire d'ajout /3

  • Créez un formulaire en respectant les conventions permettant d'ajouter un nouvel article à votre liste.
  • Liste des champs / type associé :
    • Titre de l'article
    • url de l'image / texte
    • Auteur / texte
    • Catégorie / select
      Liste des catégories
      • Dev front
      • Base de données
    • Date de publication / date
    • Résumé / textarea
  • Faire en sorte que ce formulaire apparaisse dans une modal au clic sur le bouton "ajouter un article" de l'exercice précédent

Validation du formulaire /3

  • Lier le fichier exercices_actualites.js qui se trouve dans le dossier scripts à cette page.
  • Faites en sorte de créer un nouvel item à votre liste d'articles à la validation du formulaire.
  • Faites en sorte qu'en fonction de la catégorie choisie, le design de la pastille soit différent.
  • Faites en sorte de réinitialiser le formulaire et de fermer la modal après validation.

Mettre en place un accordeon /2

  • Depuis le fichier main.js, faites en sorte que chaque exercice de cette page apparaisse via un système d'accordéon.
  • => Cacher les consignes et les contenus des exercices par défaut et faite en sorte de les faire apparaitre au clic sur le h2
  • Vous aurez besoin d'ajouter des éléments dans la page et de travailler le CSS.
